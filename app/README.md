# React + TypeScript + Vite

Iniciar localmente:

```bash
cd app
npm install
npm run dev
```
Running: http://localhost:5173


# Iniciar proyecto con Docker

Iniciar el proyecto con Docker:

```bash
docker-compose up --build
```

Finalizar el proyecto con Docker:

```bash
docker-compose down
```
