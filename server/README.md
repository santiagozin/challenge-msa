# Node JS + Express + Typescript

Iniciar localmente:

```bash
cd server
npm install
npm run dev
```
Running: http://localhost:4000


# Iniciar proyecto con Docker

Iniciar el proyecto con Docker:

```bash
docker-compose up --build
```

Finalizar el proyecto con Docker:

```bash
docker-compose down
```
